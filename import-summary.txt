Moved Files:
------------
Android Gradle projects use a different directory structure than ADT
Eclipse projects. Here's how the projects were restructured:

* AndroidManifest.xml => generalUpdateLib\src\main\AndroidManifest.xml
* assets\ => generalUpdateLib\src\main\assets\
* res\ => generalUpdateLib\src\main\res\
* src\ => generalUpdateLib\src\main\java\
